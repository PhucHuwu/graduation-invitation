# Graduation Invitation Web Application

A beautiful, modern, responsive graduation invitation card website built with Next.js and Tailwind CSS.

## Features

- **Next.js & React 19**: Modern React architecture.
- **Tailwind CSS v3**: Clean, responsive layout for Mobile, iPad, and Desktop devices.
- **Interactive Countdown Timer**: Solved hydration mismatch issues using Client-side mounting.
- **Multi-platform Calendar Integration**:
  - **Android / PC (Google)**: Link to save the event directly to Google Calendar.
  - **iPhone (Apple)**: Generate and download `.ics` file dynamically for Safari/Apple Calendar.
- **Google Maps Integration**: Easily find the location "Hanoi National University of Education".
- **Custom Local Fonts**: Self-hosted fonts (`FC-Coldwell-Bridges` & `Great Vibes`) for high stability.

## Getting Started

First, install the dependencies:

```bash
pnpm install
```

Second, run the development server:

```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Building for Production

To build the static optimized output:

```bash
pnpm build
```
